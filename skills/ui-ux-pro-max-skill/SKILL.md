# UI UX Pro Max Skill

This skill provides design intelligence for building professional UI/UX. It uses a reasoning engine to analyze project requirements and generate tailored design systems.

## Key Capabilities

- **Design System Generation**: Automatically generates a complete design system (Pattern + Style + Colors + Typography + Effects) based on the product type.
- **67 UI Styles**: Support for styles like Glassmorphism, Bento Grid, Minimalism, AI-Native UI, and more.
- **161 Industry-Specific Reasoning Rules**: Tailored advice for SaaS, Fintech, Healthcare, E-commerce, etc.
- **99 UX Guidelines**: Best practices and accessibility rules.

## How to Use

When asked to design, build, or improve a UI, invoke this skill to get a comprehensive design system recommendation.

### Example Prompts:
- "Build a landing page for my SaaS product using UI UX Pro Max"
- "Create a dashboard for healthcare analytics following UI UX Pro Max principles"

## Anti-Patterns to Avoid
- Avoid generic AI aesthetics (purple/pink gradients) unless specifically requested.
- Avoid low contrast and poor accessibility.
- Avoid inconsistent spacing and hierarchy.

## Design Checklist
- [ ] No emojis as icons (use SVG: Heroicons/Lucide)
- [ ] `cursor-pointer` on all clickable elements
- [ ] Hover states with smooth transitions (150-300ms)
- [ ] Text contrast 4.5:1 minimum
- [ ] Responsive design for 375px, 768px, 1024px, 1440px
